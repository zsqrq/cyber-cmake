◆ use_SIMPLE_RTPSParticipantDiscoveryProtocol

bool eprosima::fastrtps::rtps::BuiltinAttributes::use_SIMPLE_RTPSParticipantDiscoveryProtocol

If set to false, NO discovery whatsoever would be used. Publisher and Subscriber defined with the same topic name would NOT be linked. All matching must be done manually through the addReaderLocator, addReaderProxy, addWriterProxy methods.

◆ use_STATIC_EndpointDiscoveryProtocol

If set to true, StaticEDP based on an XML file would be implemented. The XML filename must be provided.
